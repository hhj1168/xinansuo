# 🏮 心安所寄 (Xin An Suo Ji)

> **愿你心安，所寄有应。一款融合传统文化与现代美学的数字化仪式感应用。**

[![License](https://img.shields.io/badge/License-Modified--MIT-orange.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Web-blue.svg)](https://yourusername.github.io)

「心安所寄」是一款专为追求内心宁静、探索传统文化的人群设计的数字化祈愿应用。它不仅仅是一个抽签工具，更是一场沉浸式的东方美学体验。通过精心设计的仪式感交互、唯美的神明立绘以及空灵的背景乐章，为用户提供一个在纷繁世界中与自我对话的静谧空间。

---

## ✨ 项目亮点

### 🎭 沉浸式互动仪式
- **多段式交互**：从“净手焚香”到“三跪九叩”，再到“诵经礼拜”，完整复刻传统祈福流程。
- **神明神启**：完成交互后，每位神明将降下专属的数字化“神启”寄语，实现跨越时空的心灵回响。

### 🎨 东方美学视觉体系
- **精美立绘**：收录 17 位中华传统神明（如文昌帝君、送子观音、月老等），采用统一的、具有现代艺术感的重绘风格。
- **金石美学**：界面采用玻璃拟态、鎏金质感与极简排版，适配现代审美。

### 📜 深度内容支持
- **专业签文库**：内置多位神明的专业签文衍生解读，不仅包含签诗，更有“白话详解”与“行动方案”。
- **精美海报生成**：支持一键生成超清（Retina 级别）数字化灵签海报，便于社交媒体分享与个人珍藏。

### 🎵 禅意音效系统
- **专属 BGM**：根据所选神明的不同特质，自动切换对应的禅意曲目（如《大辞大悲》、《月下仙音》等）。
- **物理反馈**：真实的钟鸣、摇铃与祈愿反馈音效。

## 🛠️ 技术栈

本项目基于经典的 Web 技术栈开发，追求轻量、高效与极致的浏览器兼容性。

- **核心**: HTML5 / CSS3（Vanilla CSS） / JavaScript (ES6+)
- **导出引擎**: `html2canvas` (用于 Retina 级别的海报渲染)
- **存储**: `localStorage` (本地化持久数据管理，保护用户隐私)
- **视觉控制**: 基於 Canvas 的粒子烟雾系统与 CSS3 滤镜动画

## 📦 快速开始

### 预览
1. 下载或克隆仓库到本地。
2. 直接在浏览器中打开 `index.html`。
3. **重要提示**：现代浏览器限制自动播放音乐，请在进入页面后点击屏幕任意位置以激活完整的音效体验。

### 部署到 GitHub Pages
1. 将所有文件上传到您的 GitHub 仓库。
2. 进入仓库 `Settings` -> `Pages`。
3. 选择 `main` 分支并点击 `Save`。
4. 访问 `https://YOUR_USERNAME.github.io/PROJECT_NAME` 即可体验。
